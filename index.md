# GST Invoice

A simple mobile app for Indian small businesses to create GST-compliant invoices.

## Features
- Create GST invoices in seconds
- Manage clients
- WhatsApp share invoices
- PDF generation
- CGST, SGST, IGST calculation
- Cloud backup

## Subscription
- **Free:** 5 invoices/month
- **Pro Monthly:** ₹79/month
- **Pro Yearly:** ₹499/year
